Question 1: What are the different ways to select an element in the DOM?

There are several ways to select elements from the DOM such as `getElementById`, `getElementsByClassName`, `getElementsByTagName`, `querySelector`, and `querySelectorAll`. Among them, `querySelector` is very flexible and works like CSS selectors to locate elements.

Question 2: What is the difference between innerHTML, innerText, and textContent?

`innerHTML` returns the HTML along with tags. `innerText` returns only the visible text, excluding hidden elements via CSS. On the other hand, `textContent` returns all text, even hidden ones, but ignores the HTML tags. Each has different use cases.

Question 3: What is event delegation in the DOM?

Event delegation is a technique where an event listener is added to a parent element instead of multiple child elements. The event is then handled through the parent when triggered by its children. This improves performance and works with dynamically added elements.

Question 4: What is event bubbling in the DOM?

Event bubbling is a behavior in the DOM where an event starts from the innermost element and propagates up to the outer elements. That means the event triggers on the child first, then its parent, and then further ancestors. It is useful in event handling.

Question 5: How do you create, add, and remove elements using JavaScript?

In JavaScript, you can create a new element using `document.createElement`. Then it can be added to the DOM using `append` or `appendChild`. To remove an element, you can use `removeChild` or simply `element.remove()`. This helps in manipulating the DOM structure.
